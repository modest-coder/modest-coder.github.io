# modest-coder

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate

# Deploy the static files to the github pages (dev-dependencies => gh-pages)
$ npm run deploy
```
